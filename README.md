# Adaptive Spectral Agro-Voltaic System (ASAS)

## Overview
ASAS is an intelligent sunlight management system for agriculture.

The system:
- Monitors temperature and light intensity
- Dynamically controls spectral filters
- Reduces harmful infrared radiation
- Enhances plant growth
- Generates renewable solar energy

## Features
- Real-time environmental monitoring
- Intelligent IR blocking
- Solar energy tracking
- Dashboard visualization
- Automated control system

## Technologies Used
- React.js
- Tailwind CSS
- ESP32
- Sensors
- Servo Motors
- Solar Panels

## Purpose
To improve crop resilience and optimize renewable energy generation using adaptive sunlight control.
